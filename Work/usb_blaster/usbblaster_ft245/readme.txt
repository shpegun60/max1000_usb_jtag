1: Створіть новий проект і додайте файли jtag_logic.vhd та Pin_Setup.tcl.
2. Встановіть jtag_logic.vhd як файл верхнього рівня.
3. Виконайте PIN-код, вказаний у файлі сценарію Pin_Setup.tcl.
4. Вичерпне, завантажте.
5. Запустіть CDM_Setup.exe, щоб встановити драйвер FT245B
6. Запустіть MProg3.0_Setup.exe, щоб записати revb.ept в EEPROM.
7. Тест.
